# 🚐 Van Life App

A responsive van rental application built with React and Vite that allows users to browse available vans, explore details, and plan their next adventure.

---

## ✨ Features

- Browse available camper vans
- View van details and pricing
- Responsive modern UI
- Dynamic routing and navigation
- Mobile-friendly design

---

## 🛠️ Built With

- React
- Vite
- JavaScript
- CSS

## 🚀 Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Once the server is running, open your browser and navigate to:

```text
http://localhost:5173
```
or 

Once the server is running, open the local development URL shown in your terminal.

---

## 📦 Build for Production

```bash
npm run build
```
